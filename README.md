##How to compile the epub
You can compile the epub by writing these instructions into Terminal:

```
$ zip -X filename.epub mimetype
$ zip -rg filename.epub META-INF -x \*.DS_Store
$ zip -rg filename.epub OEBPS -x \*.DS_Store
```
